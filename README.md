# 🤖 AI SQL Query Generator

An AI-powered SQL Query Generator built using **Streamlit** and **Google Gemini API**.

This application converts natural language questions into SQL queries using AI.

---

# ✨ Features

✅ Generate SQL queries using AI  
✅ Simple and clean Streamlit UI  
✅ Powered by Google Gemini API  
✅ Fast query generation  
✅ Beginner-friendly project  

---

# 🛠️ Technologies Used

- 🐍 Python
- 🎨 Streamlit
- 🤖 Google Gemini API
- 📦 google-generativeai library

💻 PowerShell
